منصة أسرار عيد الوكرة — النسخة النهائية

طريقة التشغيل:
1. فكّي ضغط الملف.
2. افتحي index.html.
3. تعمل المنصة على الجوال والآيباد واللاب توب.

للنشر كرابط:
- ارفعي محتويات المجلد إلى Netlify أو GitHub Pages.
- بعد فتح الرابط من الآيفون: مشاركة > إضافة إلى الشاشة الرئيسية.

محتويات المنصة:
- شاشة افتتاحية فاخرة.
- شعار المدرسة وصورة سلامة.
- خريطة الوكرة التفاعلية.
- خمس محطات متسلسلة.
- بنك أسئلة متنوع في اللغة العربية والهوية الوطنية القطرية.
- نقاط وأختام وأوسمة.
- جواز إلكتروني للطالبة.
- لوحة أبطال للفرق.
- مؤثرات صوتية واحتفالية.
