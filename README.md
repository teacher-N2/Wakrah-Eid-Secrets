# Wakrah Eid Secrets

مشروع جاهز للنشر على GitHub Pages.

## الملفات
- index.html
- css/style.css
- js/app.js
- js/questions.js
- assets/ جميع الصور والبوابات والشعار وشخصية سلامة

## النشر
ارفعي كل الملفات والمجلدات كما هي في جذر المستودع:
Wakrah-Eid-Secrets

ثم:
Settings → Pages → Branch: main → Folder: /root → Save
